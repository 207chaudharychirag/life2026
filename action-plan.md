## Phase 0 — June 10–24: Test Both Lanes (2 weeks)

**Theme: Don't choose blind. Sample both cheaply, then commit to ONE.**

### Lane A — Marketing/Comms test (~1 hr/day)
- [ ] Start **Google Digital Garage: Fundamentals of Digital Marketing** (free, ~40 hrs total — just do the first modules)
- [ ] Write 3 LinkedIn posts about anything business-related you find interesting
- [ ] Pick one brand you like and write a 1-page "what I'd improve about their marketing" mini case study

### Lane B — Analyst test (~1 hr/day)
- [ ] Do a free intro SQL course (Khan Academy "Intro to SQL" or sqlbolt.com — both free, browser-based)
- [ ] Learn 5 core Excel/Google Sheets skills: VLOOKUP/XLOOKUP, pivot tables, charts, IF formulas, sorting/filtering
- [ ] Take any public dataset (e.g. Kaggle) and answer 3 questions about it in a spreadsheet

### Decision day — June 24. Ask honestly:
1. Which one made me lose track of time, and which one felt like homework?
2. Which output am I prouder of — the case study/posts, or the data analysis?
3. Could I do this 8 hours a day?

→ **Write the answer at the bottom of this file. Then drop the other lane completely.**

## Phase 1 — June 24 → September: Build the Skill Stack + Portfolio

**Theme: One lane, 8–12 weeks, real proof of work. Applications start once there's something to show (≈ week 3 of the lane).**

### If Marketing/Comms lane:
**Skill stack (free/cheap, in order):**
1. Finish Google Digital Garage Fundamentals of Digital Marketing (certificate)
2. HubSpot Academy — Content Marketing + Social Media certifications (free)
3. Google Analytics 4 certification (free)
4. Basics of Canva, Mailchimp, Meta Ads Manager (learn by doing, not courses)

**Portfolio (this is what gets interviews):**
- [ ] 3 marketing case studies (real brands, your analysis + recommendations)
- [ ] Run marketing for ONE real thing for 4+ weeks: a local business for free, a community group, or your own small project — screenshots + results = portfolio gold
- [ ] LinkedIn: post weekly about what you're learning (visible proof + networking in one move)

**Apply to:** Marketing Coordinator, Marketing Assistant, Content Coordinator, Communications Assistant, Social Media Coordinator

### If Analyst lane:
**Skill stack (in order):**
1. Excel to advanced level (pivot tables, lookups, dashboards)
2. SQL properly (Mode SQL tutorial / Coursera)
3. Power BI (free desktop version) — build dashboards
4. Optional: Google Data Analytics Certificate (Coursera) as a structured spine

**Portfolio:**
- [ ] 3 analysis projects on public datasets (Kaggle), each with a written summary of findings — retail datasets are smart (you can talk about Woolworths context in interviews)
- [ ] 1 Power BI dashboard published to your portfolio

**Apply to:** Junior Business Analyst, Operations Analyst, Data Analyst (entry), Reporting Analyst

### Application targets (either lane)
- [ ] Resume rewritten around the chosen lane (MBA + portfolio + Woolworths = reliability, customer understanding, retail operations exposure)
- [ ] 10 tailored applications/week from week 3 of the lane onward
- [ ] 20 min/day networking on LinkedIn (connect with people in the target role, comment, ask one genuine question)
- [ ] Track everything in the table at the bottom

### Also in Phase 1
- [ ] Assignment 2 — complete and submit (by mid-July at the latest)
- [ ] Interview prep from August: STAR stories, role-specific basics, mock interviews
- [ ] One social activity/week — don't isolate during the grind
